## 📌 About this project

This tool automates progression tasks in Dead by Daylight. It assists players by managing Bloodpoints, unlocking character perks, and managing cosmetic skins. This software runs as an external process. It does not modify game files directly. This approach helps maintain account security while providing automation features for faster character leveling.

The main interface displays four core modules. Select the check-box next to each feature to enable it.

### Bloodpoint Automation
This module manages your Bloodpoints. It detects your current balance and processes items from the Bloodweb automatically. Set the limiters in the settings menu if you wish to stop at a specific amount.

### ESP Wallhack
This feature renders an overlay on your screen. It displays the locations of survivors, killers, and generators through surfaces. You can toggle this overlay using the F1 key while inside a match.

### Aimbot Assist
When enabled, this modules helps align your character view during interactions. It adjusts tracking speed based on mouse input. Use this for smoother performance during skill checks or long-range interactions.

### Perk and Skin Unlocker
This tool unlocks cosmetics that you already own in your account database. It handles the server-side requests required to display these items in the lobby and during gameplay.

## 🛡️ Staying safe

External tools carry risks when used in online games. While this trainer uses external memory reading, adhere to these practices:

*   Do not use the trainer on public accounts if you worry about account bans.
*   Limit the speed of bloodweb automation to mimic human input patterns.
*   Disable automated features during high-traffic events to reduce suspicion.
*   Keep the software updated. Old versions cause errors and increase detection risk.

## ❓ Frequently asked questions

### Will this software change my game files?
No. The trainer reads game memory. It never injects code or modifies the game's executable files. 

### Why does my antivirus flag this file?
Tools that read process memory often trigger antivirus software because they mimic the behavior of sensitive applications. This is a false positive. You can add the folder as an exception in your antivirus settings.

### Do I need to be in the game menu?
Yes. Launch the game first. Once the game reaches the main menu, open the trainer. If the software does not detect the game, click the Refresh icon in the trainer menu.

### How do I report a bug?
If you experience a crash or a feature stops working, open a new issue in the Issues tab of this GitHub repository. Include your version number and a screenshot of the error message to help with troubleshooting.
